About CertGenius

CertGenius is a comprehensive certificate management system designed for educational institutions to streamline the process of certificate requests, approvals, and delivery.

Fast Processing

Reduce certificate processing time from days to minutes with our automated workflow.

Secure & Reliable

Your data is protected with industry-standard security measures and encryption.

Track & Monitor

Real-time tracking of your certificate requests with status updates at every stage.

System File Structure

All connected files in your CertGenius system

Core System Files
  • db_config.php - Database configuration
  • init_db.sql - Database initialization
  • style.css - Main stylesheet
  • js_app.js - JavaScript functionality
Functional Files
  • student_login.php/html - Student authentication
  • admin_login.php/html - Admin authentication
  • student_home.php - Student dashboard
  • admin_dashboard.php - Admin dashboard
  • student_requests.php - Request history
  • create_student.php/html - Create student accounts
  • approve_request.php - Approve certificates
  • reject_request.php - Reject certificates
  • logout.php - System logout
